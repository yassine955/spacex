# SpaceX - Webapp

I made this when i was bored...

DEMO: http://spacexyassine.herokuapp.com/

## Installation

In current directory

```bash
npm i
```
Go into the client folder

```bash
npm i
```
## Usage

Go back to the root folder:

```bash
npm run dev
```
